# متجر فيب إلكتروني (Headless E-commerce)

واجهة أمامية سريعة ومبنية خصيصاً للسوق الكويتي باستخدام بنية Headless. تعتمد على Next.js لعرض واجهة المستخدم، وووردبريس (WooCommerce) كمنصة خلفية لإدارة الطلبات والبيانات.

## المميزات التقنية
* **أداء فائق:** تصميم يعتمد على HTML/CSS نظيف بالكامل بدون مكتبات واجهة مستخدم (UI Libraries) خارجية لضمان سرعة تحميل فورية وتوافق ممتاز مع SEO.
* **تصميم متجاوب وRTL:** دعم كامل للغة العربية من اليمين لليسار، مع تحديد أقصى عرض للحاوية (Container) بـ 1600 بكسل.
* **تصفح مرن:** عرض سريع لجميع تصنيفات المتجر (سحبات جاهزة، بودات، نكهات، كويلات، تانكات) عبر استعلامات GraphQL.

## التقنيات المستخدمة
* **الواجهة الأمامية:** Next.js (App Router).
* **الواجهة الخلفية:** WordPress + WooCommerce + WPGraphQL + WooGraphQL.
* **الاستضافة:** Vercel.

## المتغيرات البيئية (Environment Variables)
لربط الواجهة الأمامية بالخلفية بنجاح، قم بإنشاء ملف `.env.local` في الجذر الرئيسي للمشروع وأضف الرابط التالي (أو أضفه في إعدادات Vercel):
