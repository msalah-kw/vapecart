# Project Overview
- **Project Type:** Headless E-commerce Store for Vape Products.
- **Target Market:** Kuwait.
- **Currency:** KWD (دينار كويتي).
- **Backend:** WordPress (WooCommerce) + WPGraphQL + WooGraphQL.
- **Frontend:** Next.js (App Router).
- **Shipping Logic:** Must include Kuwaiti area names (e.g., Hawally, Salmiya, Al-Ahmadi) for precise shipping zones calculation.
- **Main Categories:** سحبات جاهزة, بودات, نكهات, كويلات, تانكات, زقاير تيريا, زقاير هييتس.
